## Codebase Patterns
- Avatar component in `apps/web/src/components/Avatar.tsx` accepts `size` prop: "sm" (32px), "md" (40px), "lg" (48px)
- Friends page uses `<Avatar>` for friend cards, search results, and pending request cards
- NavBar icon active color is a hardcoded hex in SVG stroke attributes
- Tailwind responsive breakpoints: `md:` prefix for desktop layout overrides
